# Dcard 貼文草稿 (工作板 or 科技板)

---

**每天被 LINE 群組轟炸？我寫了一個 AI 幫你整理**

先說結論：我做了一個開源工具，每天早上自動幫你把所有 LINE 訊息整理成一頁摘要。誰在找你、什麼事要回、什麼可以不看。30 秒看完。

為什麼做這個？

因為我每天早上打開 LINE 就焦慮。20 幾個群組，一堆未讀。看之前不知道哪些重要，看的時候要花 30 分鐘，看完之後發現 90% 是早安圖和貼圖。

LINE 不可能幫你解決這個問題。他們的營收來自你的焦慮——你越常打開 APP，他們廣告曝光越多、官方帳號推播越多人看。

所以我自己做了一個。它長這樣：

```
📋 LINE 摘要 | 03/30 08:00

📌 需要回覆 (3)
━━━━━━━━━━
【行銷部群組】
→ Amy 問素材好了嗎 (07:50)
→ 主管 Kevin：下午 3 點客戶來，要準備 (07:18)

【Jenny】
→ 約週六吃新開的餐廳 (07:45)

💤 可略過
━━━━━━━━━━
3 則早安圖 · 8 則閒聊
```

摘要直接出現在你的 LINE 裡（自己跟自己聊天），不用裝新 APP。

目前需要會一點技術（Node.js + terminal），之後會做到掃 QR code 就能用。

開源免費：github.com/dans-huang/line-digest

很早期，如果你也有 LINE 焦慮，歡迎試用或留言告訴我你最痛的點是什麼。
